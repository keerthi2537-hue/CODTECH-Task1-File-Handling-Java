## Task Name
File Handling Utility

## Description
This project is developed as part of the CODTECH Internship.
It is a Java-based File Handling Utility that performs basic
file operations such as reading, writing, and modifying text files.

## Features
- Write content to a text file
- Read content from a text file
- Modify (append) content in a text file

## Technologies Used
- Java
- Eclipse IDE

## How to Run
1. Open the project in Eclipse
2. Run FileHandlingUtility.java
3. Select an option from the menu:
   - 1 to write data to the file
   - 2 to read data from the file
   - 3 to modify data in the file

## Output
The program performs file operations on a text file named `sample.txt`
and displays the results in the Eclipse console.
